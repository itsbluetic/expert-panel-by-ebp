# 전문가소환 by EBP

근거 기반(Evidence-Based Practice) 전문가 패널 토론 스킬 for [Claude Code](https://claude.com/claude-code).

주어진 주제에 대해 **관련 학문 분야의 근거를 먼저 탐색**한 후, 해당 근거에 기반한 전문가 패널 토론을 진행합니다. "전문가가 이렇다고 해서 무조건 따르지 않는다"는 원칙 하에, 모든 주장에 근거 수준 태그를 붙여 투명하게 토론합니다.

## 핵심 프로세스

1. **학문 지도** — 주제 관련 학문 분야 식별
2. **근거 지형 탐색** — 현상 이해 + 실천 방법 + 통념 경고 정리
3. **컨텍스트 수집** — 사용자 상황/제약/목표 파악
4. **전문가 패널 토론** — 6인 전문가가 근거 기반으로 토론
5. **구현 설계** — Core/Peripheral Practice 분리, 맥락 적응
6. **실천 실험 설계** — 최소 실험 + 관찰 포인트 + 반복 주기

## 설치

```bash
# 프로젝트 로컬 스킬로 설치
cp 전문가소환.md .claude/commands/전문가소환.md

# 또는 전역 스킬로 설치
cp 전문가소환.md ~/.claude/commands/전문가소환.md
```

## 사용법

```
/전문가소환 [주제]                                    → 전체 프로세스 실행
/전문가소환 [주제] --experts [전문가1, 전문가2, ...]   → 전문가 직접 지정
/전문가소환 [주제] --quick                             → 근거 탐색 생략, 빠른 토론
```

## 근거 수준 체계

| 태그 | 의미 |
|------|------|
| **[Strong]** | 메타분석, 체계적 리뷰, 복수의 재현된 실험 |
| **[Moderate]** | 복수의 RCT 또는 대규모 관찰 연구 |
| **[Emerging]** | 단일 연구, 예비 결과, 파일럿 스터디 |
| **[Expert]** | 전문가 합의, 임상 경험 기반 (근거 약함) |
| **[Contested]** | 상반된 연구 결과가 존재 |
