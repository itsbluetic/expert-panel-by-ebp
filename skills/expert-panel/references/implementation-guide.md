# Implementation Guide — 보충 자료

이 문서는 `SKILL.md` 5-6단계의 Implementation Science와 PBE Loop에 대한 심화 사례입니다. 스킬 실행에는 불필요하며, 배경 이해를 위한 보충 자료입니다.

## Implementation Science 심화

### Core vs Peripheral Practice

Implementation Science의 핵심 구분법입니다. 어떤 실천법(practice)이든 효과를 만드는 핵심 요소(core)와 맥락에 따라 변형 가능한 요소(peripheral)가 있습니다.

**실패 사례: 빌 게이츠 재단 탁월한 교사 프로젝트**
- $575M 투자, 7년간 진행
- 연구로 밝혀진 "효과적 교수법"을 표준화하여 배포
- 결과: 학생 성취도에 유의미한 변화 없음
- 원인: Core Practice의 식별은 맞았으나, 교사 개인의 맥락적 적응(Peripheral Practice)을 무시

— Kane et al., 2014; RAND Corporation, 2018

**성공 사례: Positive Deviance 접근**
- 베트남 아동 영양실조 퇴치 (Jerry Sternin)
- 같은 빈곤 환경에서 건강한 아이를 키우는 가정의 행동 관찰
- Core: 하루 4회 소량 급식 (핵심 원리)
- Peripheral: 사용하는 식재료, 조리 방식 (지역에 맞게 변형)
- 결과: 2년 내 65% 아동의 영양 상태 개선

— Pascale, Sternin & Sternin, 2010

### Adaptation Framework

Stirman et al. (2019)의 FRAME (Framework for Reporting Adaptations and Modifications to Evidence-Based Interventions):
1. **무엇을** 변형했는가 (content, context, training)
2. **누가** 결정했는가 (연구자, 실무자, 사용자)
3. **왜** 변형했는가 (문화적 적합성, 자원 제약, 사용자 선호)
4. **핵심 기능에** 영향을 미치는가

## PBE (Practice-Based Evidence) 심화

### EBP → PBE 선순환

전통적 EBP는 "연구 → 실천" 단방향입니다. PBE는 "실천 → 근거 축적 → 개인화된 근거"로 확장합니다.

```
[연구 근거] → 실천 → 관찰 → 기록 → [개인 근거 축적] → 적응 → 재실천
     ↑                                                          ↓
     └──────────── 선순환 (Practice-Based Evidence) ────────────┘
```

— Barkham & Mellor-Clark, 2003; Green & Glasgow, 2006

### N-of-1 Trial 설계

개인 수준의 근거 축적을 위한 실험 설계:
- **교대 설계**: A(개입) → B(대조) → A → B 반복
- **다중 기저선**: 여러 행동/지표를 동시에 추적, 하나씩 개입
- **시계열**: 개입 전후의 지표 추이 비교

스킬의 6단계 MVE(Minimum Viable Experiment)는 이 N-of-1 설계의 간소화 버전입니다.
